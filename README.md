# 🎓 StudyForge AI – AI Powered Student Success Platform

## Overview

StudyForge AI is a multi-agent AI-powered student success platform designed to help students improve their career preparation journey through intelligent guidance and personalized recommendations.

The system analyzes resumes and generates customized learning roadmaps, interview preparation questions, project suggestions, and career opportunities in one place.

---

## Features

✅ AI Resume Analysis
✅ Personalized Learning Roadmap
✅ Interview Question Generator
✅ Project Recommendations
✅ Career Opportunity Suggestions
✅ Resume Upload Support
✅ Interactive Student Dashboard
✅ AI Generated Report

---

## Tech Stack

* Python
* Streamlit

## Project Architecture

Resume Upload → Resume Analysis Agent → Skill Detection → Learning Roadmap → Interview Preparation → Growth Recommendations

## Installation

Clone repository:

```bash
git clone YOUR_REPOSITORY_LINK
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
streamlit run app.py
```
